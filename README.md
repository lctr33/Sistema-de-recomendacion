# Sistema de recomendación

Repositorio para el proyecto de Minería de Texto: Sistema de recomendación de libros.

Qué incluir en el repo:
- Código fuente bajo `sistema_recomendacion/src/`.
- Notebooks principales en `sistema_recomendacion/notebooks/`.
- Documentación en `sistema_recomendacion/docs/`.
- Archivos de configuración (requirements, setup, scripts).

Qué NO incluir:
- Datasets crudos o procesados grandes (`data/raw/`, `data/processed/*.npz`, `*.joblib`, `*.csv`).

Cómo subir:
1. Crear un repositorio en GitHub llamado `Sistema-de-recomendacion` (sin espacios, o ajusta según prefieras).
2. Añadir remote y push:

```bash
git remote add origin https://github.com/<tu-usuario>/Sistema-de-recomendacion.git
git push -u origin main
```

Si quieres, puedo generar también un `requirements.txt` y un `LICENSE`.
