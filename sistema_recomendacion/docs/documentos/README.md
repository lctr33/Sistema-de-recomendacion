# Documentos formales

Guarda aquí los documentos en formato PDF o Markdown que acompañarán cada etapa:

- `etapa1_baseline.md`: análisis, preprocesamiento, representaciones y baseline.
- `etapa2_modelo_alternativo.md`: primer modelo mejorado.
- `etapa3_mejoras.md`: segunda aproximación con mejoras.
- `infraestructura.md`: configuración de hardware/software empleado (opcional pero recomendado para reproducibilidad).
- `acuerdos.md`: registro de acuerdos del equipo.

Recuerda exportar las versiones finales a PDF para su entrega oficial y conservar las fuentes editables en este mismo directorio.
