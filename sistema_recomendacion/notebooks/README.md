# Notebooks del proyecto

Organiza los cuadernos de Jupyter por etapa para seguir la rúbrica:

- `etapa1/`: análisis del problema, preprocesamiento, representaciones y baseline.
- `etapa2/`: primera aproximación alternativa y evaluación.
- `etapa3/`: segunda aproximación con mejoras y evaluación final.

Convención sugerida: prefijos numéricos para mantener el orden cronológico (`01_`, `02_`, etc.).

Asegúrate de:
- Mantener celdas limpias y con descripciones claras.
- Guardar versiones HTML o PDF de respaldo en `reports/` si se requiere.
- Referenciar los notebooks en los documentos formales para trazar la evidencia.
